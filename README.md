# 🌙 Neovim Config - MacWard 1th (Javi)

Bienvenido a mi configuración personal de Neovim. Este entorno está construido desde cero en Lua, priorizando la ligereza, la velocidad y la comprensión de cada línea de código. 

## 🛠️ Stack y Herramientas

* **Sistema Operativo:** macOS
* **Gestor de Plugins:** [lazy.nvim](https://github.com/folke/lazy.nvim)
* **Tema Visual:** TokyoNight
* **Explorador:** Nvim-Tree
* **Sintaxis:** Treesitter (Resaltado inteligente)

## 🚀 Instalación rápida

Si por algún motivo necesito clonar esta configuración en otra máquina, estos son los pasos:

1. Asegurarse de tener instalado Homebrew, Git y las herramientas de Apple (`xcode-select --install`).
2. Instalar Neovim: `brew install neovim`
3. Clonar este repositorio en la carpeta correcta:
   ```bash
   git clone [https://github.com/v-Erios/mis-dotfiles.git](https://github.com/v-Erios/mis-dotfiles.git) ~/.config/nvim
